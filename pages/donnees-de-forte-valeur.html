Les données de forte valeur

Cette page recense et met en avant les données de forte valeur comme définies par la Commission européenne.

Les données de forte valeur correspondent aux "documents détenus par un organisme du secteur public, dont la réutilisation est associée à des bénéfices importants pour la société, l'environnement et l'économie". Elles sont mises à disposition gratuitement, avec un minimum de restrictions légales et techniques afin d'augmenter leur potentiel de réutilisation et leur impact.

Réutilisations de données

A partir de ces données géospatiales, des solutions ont été conçues pour permettre de :

  • Connaître l’environnement associé à un lieu (Aux Alentours par MAIF ; iudo.app) ;
  • Connaître le prix de vente des biens immobiliers (Explorateur des données de valeur foncière ; Outil de Meilleurs Agents).
  • Connaître et partager les parcelles conduites en agriculture biologique ;
  • Créer des couches cartographiques à partir du Code Officiel Géographique et des données de l'IGN ;
  • Consulter et visualiser ses obligations légales de débrouissaillement ;

D'autres usages ont été développés à partir de ces données. Il est possible d'en découvrir davantage en parcourant les réutilisations référencées sur data.gouv.fr.

Réutilisations de données

A partir de ces données météorologiques, des solutions ont été conçues pour permettre de :

  • Visualiser les données (ClimaViz ; MétéoViz)
  • Faciliter l'exploitation et la manipulation des données (La Dalle ; Climate Accessibility)
  • Créer un score de risque de feu de forêt ;
  • Observer l'évolution de phénomènes météorologiques et comparer des indicateurs dans le temps ;
  • Connaître les conditions météorologiques lors d'une intervention agricole ou pendant la durée d'une culture ;
  • Etudier le phénomène de baisse de l'épaisseur du manteau neigeux en haute montagne ;
  • Anticiper la capacité de production d'énergies renouvelables ;
  • Estimer la vitesse maximale de vent enregistrée à une adresse en France.

D'autres usages ont été développés à partir de ces données. Il est possible d'en découvrir davantage en parcourant les réutilisations référencées sur data.gouv.fr.
