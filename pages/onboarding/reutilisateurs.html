Comment exploiter les données ouvertes ?

Vous souhaitez trouver une information, analyser un sujet, créer de nouveaux produits et services, nourrir vos travaux journalistiques ou de recherche, etc. ? Les données ouvertes sur data.gouv.fr pourraient vous être précieuses !

Utiliser les données ouvertes

Qui que vous soyez, vous pouvez exploiter les données disponibles sur data.gouv.fr pour réaliser votre projet. La réutilisation des données est libre et gratuite. En savoir plus sur les conditions de réutilisation.

L'équipe de data.gouv.fr a conçu un guide qui vous fournit les éléments essentiels pour vous permettre d'exploiter des données ouvertes.
